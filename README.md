# Menu Rekomendasi Diskrit

Deskripsi
- Proyek ini berisi program Java bernama `MenuRekomendasiDiskrit` yang menyediakan menu rekomendasi untuk topik diskrit (file utama: `MenuRekomendasiDiskrit.java`).

Prasyarat
- Java JDK 11 atau lebih baru terinstal.

Kompilasi
1. Buka terminal di folder proyek (tempat `MenuRekomendasiDiskrit.java`).
2. Jalankan:

```bash
javac MenuRekomendasiDiskrit.java
```

Menjalankan
- Setelah kompilasi sukses, jalankan:

```bash
java MenuRekomendasiDiskrit
```

Catatan
- Jika file berada dalam package, jalankan sesuai struktur package (sesuaikan perintah `javac`/`java`).
- Untuk menjalankan di IDE (IntelliJ/Eclipse/VS Code), buka proyek sebagai Java project dan jalankan kelas `MenuRekomendasiDiskrit`.
